# 测试任务编排

> 除了手动编写测试脚本，用户也可以通过 `AngusTester应用` → `场景` → `创建场景` → 可视化编排 → 自动生成测试脚本，其生成的测试脚本与以下规范是一致的。

## 核心价值
- **零编码测试设计**：通过可视化界面构建完整测试流程  
- **多协议支持**：覆盖主流协议测试需求  
- **脚本复用**：可视化生成的脚本与手工编写脚本完全兼容  
- **灵活扩展**：支持跨协议组合测试场景
