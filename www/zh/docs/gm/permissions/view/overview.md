# 查看授权

> **查看授权** 允许管理员全面审计用户、部门或用户组的权限分配情况，确保系统权限设置符合安全规范与合规要求。通过此功能，您可以：
> - 精确掌握权限分配关系
> - 验证权限策略应用效果
> - 识别异常权限配置
> - 满足合规审计需求

> **核心价值**：实现权限管理的透明化和可视化，保障系统安全运行。

## 核心功能

| 特性 | 说明 | 业务价值 |
|------|------|---------|
| **权限溯源** | 显示权限获取途径（直接授权/继承） | 明确责任归属 |
| **状态标识** | 实时显示权限生效状态 | 识别过期/异常权限 |
| **冲突检测** | 自动标记冲突权限项 | 预防越权访问 |
| **跨系统视图** | 展示用户在多个系统的权限 | 全面掌握权限分布 |
