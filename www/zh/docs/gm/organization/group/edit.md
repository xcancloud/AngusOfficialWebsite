# 编辑组

> **编辑组** 用于修改用户组的基础配置与属性，确保组设置与当前业务需求同步。编辑操作将实时更新组权限与成员关系。

## 操作指南

### 1. 进入编辑界面
1. 登录系统控制台
2. 导航至 `AngusGM → 组织人员 → 组`
3. 在组列表中：
    - 右键点击目标组
    - 选择 **「编辑组」**

### 2. 修改组信息
可调整以下字段：
- **必填项**：名称
- **可选项**：描述、组标签

> ⚠️ **命名规范**：
> - 组名称全局唯一
> - 建议不输入特殊字符：`/ \ : * ? " < > |`

### 3. 保存变更
1. 点击 **「保存」** 提交修改
2. 系统提示：**"组信息修改成功"**
3. 自动刷新组详情页
