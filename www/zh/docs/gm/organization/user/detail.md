# 查看用户信息

> **查看用户信息** 是指应用系统中，查看用户查看个人或组织成员的账户信息，如账户设置、权限等。

## 操作指南

### 方式一：通过全局管理查看（管理员）
1. 登录系统控制台
2. 访问 `AngusGM → 组织人员 → 用户`
3. 在用户列表中执行：
    - **查看他人信息**：点击目标用户 **姓名**
    - **查看自身信息**：点击当前用户行右侧 **「查看详情」**

### 方式二：通过个人中心查看（所有用户）
1. 在任何系统页面，点击右上角用户 **头像**
2. 选择下拉菜单中 **「个人中心」**
3. 系统自动跳转至当前用户详情页
